Lesson 1: The Basics

El Web

The web is made up of the following basic parts:

  1. Client
  2. Internet
  3. Servers
  4. HTTP - HyperText Transfer Protocol
  5. HTML - HyperText Markup Language

Clients communicate with servers through the internet through the protocol known as HTTP. It is the foundation of data communication for the World Wide Web.
The server will then find the HTML document the client has requested, and send it back. A web browser then interprets the HTML document and displays it on screen.

HTML for Dummies

HTML is made up of elements. An element is usually made up of a start and end tag, with content in the middle. It is a code sandwich.
Elements can be nested. In fact, they always are.

Span vs. Div

The span and div tags serve a similar purpose: they are both used to cordon off a section of the text so you can add styling to it.
The difference between the two is that span is an inline element and div is a block element.
Inline elements are able to be manipulated within the structure of a flowing text. For example if I wanted to make a specific word in the paragrah a different color, span allows me to localize all styling to as few characters as I want.
Block elements cordoned off using div allows entire sections (and sections within sections) to be styled together.
